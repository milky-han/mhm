# Milky Han Mono<br>星汉等宽 / 星漢等寬 / 星漢等幅 / 성한 고정폭

Milky Han Mono, is not only a duospaced CJK programming font family, but also a framework to [create your programming font](doc/customization.md).

> 日月之行，若出其中。星漢燦爛，若出其裏。
